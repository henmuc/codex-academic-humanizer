# Style Control

Use this reference when the user specifies revision strength, provides a target writing sample, or asks for consistent terminology across a longer academic text.

## Revision Intensity

### Light

Use when the user wants minimal intervention, proofreading, or reviewer-safe edits.

- Keep most sentence boundaries.
- Remove obvious AI-like wording and awkward grammar.
- Improve local cohesion only where needed.
- Avoid changing paragraph order or rhetorical structure.

### Medium

Use by default.

- Rewrite sentences that sound generic, repetitive, or mechanically translated.
- Improve paragraph flow while preserving paragraph function.
- Replace empty academic cliches with source-grounded phrasing.
- Keep the author's claims and evidence unchanged.

### Deep

Use when the user asks for a more natural, polished, or publication-ready rewrite.

- Recast sentence structure more freely.
- Merge or split sentences for argument clarity.
- Improve paragraph-level progression.
- Still preserve all facts, references, scope conditions, and claim strength.

## Target Style Samples

When the user provides a style sample, infer reusable features:

- Voice: first-person plural, impersonal, active, passive, or mixed.
- Density: compact, explanatory, highly technical, or reader-friendly academic.
- Hedging level: cautious, moderate, or assertive.
- Sentence rhythm: short and direct, long and syntactic, or mixed.
- Transition habits: explicit logical connectors, implicit continuity, or section-level signposting.
- Terminology habits: English terms retained, Chinese translations used, or bilingual parentheses.

Do not copy unique wording, facts, examples, citations, or claims from the sample. Treat it as a style guide, not source material.

## Terminology Control

For long documents, maintain an internal terminology map:

- Original term -> preferred revised term.
- Abbreviation -> full form.
- Chinese term -> English term if used.
- Dataset/model/method names that must not be translated.

Repeat technical terms consistently. Do not use synonyms just to avoid repetition when the term names a fixed construct.

## User Preference Defaults

If the user does not specify preferences:

- Use Medium intensity.
- Keep the original person and voice.
- Keep citation style unchanged.
- Keep section headings unchanged.
- Use dual output: clean revision plus comparison notes.
